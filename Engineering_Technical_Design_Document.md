Engineering & Technical Design Document
Architecture Overview
The system is a modern web application built with a React frontend, Python FastAPI backend, and
PostgreSQL database.
Technology Stack
• Frontend: React with TypeScript
• Backend: Python FastAPI
• Database: PostgreSQL
• Hosting: Linux VPS (Azure-ready)
Security
• Multi-factor authentication
• Role-based access control
• Audit logging
• Future Azure AD SSO integration
Scalability
The application is designed to evolve from single-tenant to multi-tenant without architectural
changes.
